# Schema 空间设计分册（2026-07-11 立册，规范级）

状态：骨架已拍板，细则由 SCHEMA-SPEC-1 工单按本骨架填写、架构过目后转规范。目的：**使大多数 schema 设计如出一手**——词表（docs/57）定"有哪些原语"，本分册定"每个原语长什么样"；新 renderer 按册盖章生产，禁止每次引入新设计。

## 一、嵌套层级模型（五级封闭）

```
PreviewHost（L1 浮面，唯一白卡）
└─ section（版面分段：标题 510 + 分隔线，无卡壳）
   └─ 语义块（semantic-list 行 / 主从列表 / 表格——单层圆角容器上限，法理之线居左贴边 + 12px gutter）
      └─ 行（dense 13px/1.5，选中仅 bg.selected 无外框）
         └─ 字段（mono 位：编号/日期/金额 tabular-nums；信源角标固定行尾；溯源 hover 展开引语）
```

嵌套上限即此五级；更深语义一律弹 L2 或路由至从属视图，禁止第六级视觉嵌套。

## 二、通行凡例（引用既有拍板，不另立）

- 排版：RP-2.3 dense 档（13/12）+ 文书阅读 15px 不变量；指标数字 metric 28px 仅 statcard。
- 控件：RP-2.6 control.* 全套（28/32 高、400/510 二元、icon 16、gap 6）；三区高 44/40/36。
- 色彩：语义色白名单五态仅审阅语义；color.data.* 物理隔离；轨道与骨架灰阶。
- 绘制：SVG 全原生（P-4 管线）；图表族按 docs/57 token 增量四组。
- 动效：数据区静止；等待态 = #26 三态闭环形制；0ms 硬切。
- 固定位形：法理之线（左贴语义块边）、信源角标（行尾）、门禁卡（承重卡形制不减重）、滚动轨（宿主右缘 2px）。

## 三、落地纪律

1. **先回灌后新增**：既有五工作面（时间线/图谱/矩阵/修订/起草）按本册对齐，是第一批消费方与试金石；对齐中发现册内缺漏 → 补册再改码。
2. 新 renderer 提案必须逐条声明"消费本册哪些凡例"，出现册外样式即打回。
3. 分册细则（各原语的规格图与间距标注）由 SCHEMA-SPEC-1 填写，架构过目后生效。
