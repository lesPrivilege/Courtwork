# 民事答辩状

答辩人（被告一）：起云智能装备（虚构）有限公司
法定代表人：岑瑞霖
诉讼代理人：练明轩，恒屿律师事务所律师

针对原告临江精铸科技有限公司提起的诉讼，答辩人答辩如下：

**一、涉案设备存在质量瑕疵，答辩人依法有权拒绝或暂缓支付验收款**
2024年12月10日验收当日，答辩人现场人员即发现熔炼炉温控系统运行异常，此事实在2024年12月20日双方协商会议纪要中已有明确记载，验收单载明"验收合格"与实际情况不符，答辩人有权就此提出异议并暂缓付款。

**二、原告催告函盖章主体与合同签约主体不一致，催告效力存疑**
原告2025年1月10日发出的催告函（一）落款加盖"临江精铸集团有限公司"印章，而合同卖方为"临江精铸科技有限公司"，两者系不同法律主体，该催告函不能视为合同相对方的有效催告，原告主张的违约金起算时点应予重新审查。

**三、原告部分债权已转让第三方，原告主体资格及诉请金额存疑**
原告已于2025年3月20日将本案项下应收账款2,660,000元转让予华瑞商业保理（云章）有限公司并通知答辩人，原告是否仍有权就已转让部分主张权利存疑，请求追加华瑞商业保理为本案第三人。

**四、违约金约定畸高，请求依法予以酌减**
合同约定逾期付款每日按未付金额1%计收违约金，年化利率远超法律保护上限，显失公平，请求人民法院依法酌减。

**五、实际发货主体与合同主体不一致**
送货单载明发货单位为"临江精铸（云章）装备有限公司"，与合同卖方"临江精铸科技有限公司"不一致，交付义务的实际履行主体存疑。

综上，请求驳回原告全部或部分诉讼请求。

答辩人：起云智能装备（虚构）有限公司
2025年6月2日
