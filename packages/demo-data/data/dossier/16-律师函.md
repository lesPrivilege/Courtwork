# 律师函

致：起云智能装备（虚构）有限公司

江城明澈律师事务所受临江精铸科技有限公司（下称"委托人"）委托，就贵司拖欠合同编号LJJZ-CY-2024-0817项下验收款项一事，发出本律师函。

经了解：委托人与贵司于2024年8月17日签订《精密铸造生产线设备采购合同》，委托人已依约完成设备交付并于2024年12月10日经双方验收合格，贵司应于2024年12月25日前支付验收款2,280,000元，但贵司迄今未付。委托人已于2025年1月10日、2025年3月5日两次发函催告，贵司仍未履行付款义务。

现依据委托人指示，正式函告贵司：请于收到本函之日起10日内向委托人指定账户支付上述验收款项及相应违约金，逾期委托人将依法提起诉讼，并要求贵司承担因此产生的诉讼费、律师费等全部合理支出。

特此函告，请贵司高度重视。

江城明澈律师事务所
经办律师：柏景行
日期：2025年4月2日
