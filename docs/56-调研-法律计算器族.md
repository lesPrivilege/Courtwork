## 二、A 区详述 · 民事赔偿（13 项）

### A1 人身损害赔偿计算器

**1. 案由/触发场景**：机动车交通事故责任纠纷、健康权/身体权/生命权纠纷、医疗损害责任纠纷、提供劳务者受害责任纠纷、产品责任纠纷。触发条件：卷宗内出现伤残鉴定意见书、医疗费票据、交通事故认定书、死亡证明任一文书类型（ingest 文书分类直接命中），或对话中出现赔偿清单意图。这是竞品第一入口，也是律师日常最高频的计算场景之一。

**2. 输入参数 schema 草案**

| 字段 | 含义 | 类型 | 必填 | 卷宗可抽取 |
|------|------|------|------|-----------|
| courtProvince | 受诉法院所在省（定赔偿标准） | enum(省级行政区) | 必填 | ✅ 起诉状/受理通知 |
| accidentDate | 侵权行为/事故日期（定适用标准年度与新旧解释分界） | date | 必填 | ✅ 事故认定书 |
| victimAge | 受害人年龄（残疾/死亡赔偿金年限递减） | int | 必填 | ✅ 身份信息 |
| disabilityGrades | 伤残等级（支持多处伤残，1–10 级数组） | int[] | 条件必填 | ✅ 鉴定意见书 |
| isDeath | 是否死亡案件 | bool | 必填 | ✅ |
| medicalExpenses | 医疗费（票据合计） | money | 可选 | ✅ 票据 OCR 汇总 |
| monthlyIncome | 受害人月收入（误工费基数） | money | 可选 | ✅ 工资流水/纳税记录 |
| lostWorkDays | 误工天数 | int | 可选 | ✅ 病历/鉴定 |
| nursingDays / nursingStaff / nursingDailyRate | 护理天数/人数/日标准 | int/int/money | 可选 | 部分 ✅ |
| hospitalDays | 住院天数（伙食补助） | int | 可选 | ✅ 出院记录 |
| nutritionDays | 营养期 | int | 可选 | ✅ 鉴定 |
| dependents[] | 被扶养人（年龄、与其他扶养人分摊数） | object[] | 可选 | ✅ 户籍/亲属关系 |
| liabilityRatio | 责任比例 | percent | 必填 | ✅ 事故认定书（同等/主次责映射须人确认） |
| jqxApplied | 是否交强险先行赔付（交通事故场景） | bool | 可选 | ✅ 保单 |
| appliedStandardYear | 适用标准年度（默认取一审法庭辩论终结前最新公布数） | int | 自动 | — |

**3. 法律依据**（须核验现行有效性）：《民法典》第 1179、1183 条；《最高人民法院关于审理人身损害赔偿案件适用法律若干问题的解释》（法释〔2003〕20 号，经法释〔2022〕14 号修正——**2022-05-01 起城乡标准统一，残疾赔偿金/死亡赔偿金/被扶养人生活费一律按城镇居民标准**，这是新旧分界参数）；残疾赔偿金 = 城镇居民人均可支配收入 × 20 年 × 伤残系数（60 岁以上每增 1 岁减 1 年，75 岁以上按 5 年）；丧葬费 = 受诉法院所在地上年度职工月平均工资 × 6 个月；被扶养人生活费按城镇居民人均消费支出；精神损害抚慰金依《最高人民法院关于确定民事侵权精神损害赔偿责任若干问题的解释》（法释〔2001〕7 号，2020 修正），为裁量项只出参考区间。多处伤残系数叠加规则参照《人体损伤致残程度分级》(2017) 附录及 GB/T《伤残鉴定》晋级原则〔核验：多等级伤残赔偿附加指数各地法院掌握不一，输出须标注为地区惯例区间〕。交通事故叠加交强险分项限额（2020 年调整后：死亡伤残 18 万/医疗 1.8 万/财产 0.2 万；无责 1.8 万/0.18 万/100 元）〔核验〕。

**4. 数据依赖**：各省统计局年度公布的城镇居民人均可支配收入、人均消费支出、职工年平均工资（多数省高院/公安交管每年 5–6 月发布道交赔偿标准汇总，可作二手校验源）；国家统计局年鉴。**年度更新，31 省 × 3 指标**。伤残系数表静态。交强险限额表低频（金融监管总局调整时）。

**5. 标签**：法律专属；需地区参数 + 局部裁量区间（精神抚慰金、附加指数）。
**6. 优先级：高**。使用频率极高（人损是基层法院最大案由之一）；数据维护是年度批量作业，成本可控且正是 docs/26"数据维护即场景"的样板资产。

### A2 民间借贷利息计算器

**1. 触发**：民间借贷纠纷。卷宗出现借条/借款合同/转账凭证即触发；对话中出现"利息怎么算""能主张多少利息"。竞品第二高频入口。

**2. schema 草案**

| 字段 | 含义 | 类型 | 必填 | 卷宗可抽取 |
|------|------|------|------|-----------|
| principal | 本金 | money | 必填 | ✅ 借条/流水 |
| contractDate | 合同成立日（定 4 倍 LPR 基准月） | date | 必填 | ✅ 借条 |
| disbursementDate | 实际出借日（利息起算、砍头息核验） | date | 必填 | ✅ 转账凭证 |
| agreedRate / rateType | 约定利率及口径（年/月/日息） | rate+enum | 可选 | ✅ 借条（"月息 2 分"类表述需 LLM 归一化后人确认） |
| dueDate | 约定还款日（逾期起算） | date | 可选 | ✅ |
| repayments[] | 还款记录（日期、金额、冲抵顺序） | object[] | 可选 | ✅ 流水 |
| overdueRateAgreed | 约定逾期利率 | rate | 可选 | ✅ |
| calcEndDate | 计算截止日 | date | 必填 | 默认今日 |
| upfrontDeduction | 是否存在预扣利息（砍头息，按实际出借额定本金） | bool+money | 可选 | ✅ 流水与借条差额比对 |

**3. 法律依据**（须核验）：《最高人民法院关于审理民间借贷案件适用法律若干问题的规定》（法释〔2020〕17 号）第 25 条——**司法保护上限 = 合同成立时一年期 LPR 的 4 倍**；第 28、29 条逾期利率规则；分段适用：2020-08-20 前成立的合同适用旧 24%/36% 两线规则（法释〔2015〕18 号）〔核验：新旧衔接按最高法适用时间规定分段〕；2019-08-20 LPR 机制改革前参照央行基准利率。《民法典》第 670 条（预扣利息按实际借款数额返还并计息）、第 680 条（禁止高利放贷；未约定利息视为无息，自然人之间约定不明视为无息）。已付超限利息抵扣本金规则。计算引擎须支持**多段利率 × 多笔还款 × 冲抵顺序（先息后本）**的分段流水计算，这是竞品黑盒最容易出错、我们分项明细最能建立信任的点。

**4. 数据依赖**：LPR——全国银行间同业拆借中心每月 20 日（节假日顺延）公布，**月度追加一行，历史表一次建成**（现值：1 年期 3.0%、5 年期以上 3.5%，2026-04 起维持〔核验〕）；1990–2019 央行贷款基准利率历史表（静态一次性）。维护成本全清单最低的动态数据。

**5. 标签**：法律专属；纯公式 + 查表（LPR 表）。
**6. 优先级：高**。频率极高、数据维护近零成本、分项明细差异化最显著。

### A3 非因工死亡待遇计算器

**1. 触发**：劳动争议（非因工死亡待遇纠纷）、继承案件中核算遗属待遇。卷宗出现死亡证明 + 劳动关系证据且无工伤认定时触发。
**2. schema**：province（省，必填，✅）、deathDate（死亡日期，必填，✅）、paymentYears（养老保险累计缴费年限，必填，✅ 社保记录）、status（在职/退休，必填，✅）、pensionMonths（退休人员领取待遇月数，条件必填）。
**3. 法律依据**（须核验）：《社会保险法》第 17 条；**人社部发〔2021〕18 号《企业职工基本养老保险遗属待遇暂行办法》**（2021-09-01 施行，全国统一）：丧葬补助金 = 死亡时本省上年度城镇居民月人均可支配收入 × 2；抚恤金按缴费年限 9–24 个月该收入递进（在职），退休人员按领取待遇年限递减。〔核验：各省经办口径与过渡政策差异〕。机关事业单位另循其他规定，超出 MVP 范围。
**4. 数据**：各省城镇居民月人均可支配收入（年度，与 A1 同源复用）。
**5. 标签**：法律专属；需地区参数（查表 + 简单公式）。
**6. 优先级：中**。频率中等，但数据与 A1 完全复用，边际成本近零，可随 A1 同批捎带。

### A4 一般工资计算器（加班费/经济补偿/赔偿金）

**1. 触发**：劳动争议全族（追索劳动报酬、经济补偿金、违法解除赔偿金、未休年假工资）。卷宗出现劳动合同/工资流水/解除通知即触发。竞品做成单一"工资计算器"，建议拆为三个子表共用一套参数。
**2. schema**：monthlyWage（离职前 12 个月平均应得工资，必填，✅ 流水）、serviceYears（工作年限，含 N+1 判断，必填，✅ 合同+社保记录）、terminationType（解除类型：协商/无过失/违法解除/劳动者被迫解除等，必填，✅ 解除通知——**定 N / N+1 / 2N 的关键分支，须人确认**）、cityAvgWage3x（当地社平工资 3 倍封顶线，自动查表）、overtimeHours{workday, restday, holiday}（三类加班时数，可选，✅ 考勤记录）、hourlyBase（时薪基数 = 月工资 ÷ 21.75 ÷ 8）、untakenLeaveDays（未休年假天数，可选）。
**3. 法律依据**（须核验）：《劳动合同法》第 40、46、47、87 条（N、代通知金、社平 3 倍封顶且年限最多 12 年、2N）；《劳动法》第 44 条（加班费 150%/200%/300%）；劳社部发〔2008〕3 号（月计薪天数 21.75）；《职工带薪年休假条例》及实施办法（未休年假 300% 含正常工资，即另付 200%）。〔核验：封顶规则仅适用于高收入者的"双封顶"口径，各地仲裁掌握差异〕。
**4. 数据**：各统筹地区上年度社平工资（年度，与 A11 部分复用；只为封顶线服务，可先只维护重点城市）。
**5. 标签**：法律专属；纯公式 + 地区封顶参数。
**6. 优先级：高**。劳动争议是律所与法务最高频领域之一；参数几乎全部可从卷宗抽取，是"LLM 填表员"范式演示性最好的一张表。

### A5 迟延履行期间债务利息计算器

**1. 触发**：执行阶段/申请执行前核算。卷宗出现生效判决书/调解书 + 履行期届满即触发；对话出现"迟延履行金""加倍利息"。
**2. schema**：judgmentDebt（生效文书确定的金钱债务，必填，✅ 判决主文）、generalInterestClause（文书载明的一般债务利息条款，可选，✅ 判决主文）、performanceDeadline（履行期届满日，必填，✅ 判决书 + 生效日推算）、payments[]（已履行记录，可选，✅）、calcEndDate（截止日，必填）。
**3. 法律依据**（须核验）：《民事诉讼法》第 264 条（历次修正后条号变动，2023 修正文本〔核验条号〕）；**《最高人民法院关于执行程序中计算迟延履行期间的债务利息适用法律若干问题的解释》（法释〔2014〕8 号）**：加倍部分债务利息 = 未清偿的生效文书确定的除一般债务利息外金额 × **日万分之一点七五** × 迟延期间；一般债务利息按生效文书确定的方法继续计算；分别计算、部分清偿冲抵顺序（第 4 条）。
**4. 数据依赖**：无——费率写死在司法解释里，修法才变。
**5. 标签**：法律专属；纯公式。
**6. 优先级：高**。执行案件量巨大、公式完全确定、零数据维护，是**第一批里成本最低的一项**。

### A6 违约金计算器

**1. 触发**：合同纠纷（买卖、借款、租赁、建工）。卷宗合同文本含违约金条款即触发；与 S5 合同对比场景天然联动（条款抽取复用）。
**2. schema**：contractAmount（合同标的额）、penaltyClause（违约金条款原文 + 归一化：固定额/日千分比/月百分比，✅ 合同 OCR，**须人确认归一化结果**）、actualLoss（可证明损失，可选，裁量锚点）、breachStartDate/EndDate、paidAmount。
**3. 法律依据**（须核验）：《民法典》第 585 条（约定违约金过分高于损失可请求调减）；**《最高人民法院关于适用〈民法典〉合同编通则若干问题的解释》（法释〔2023〕13 号）第 65 条**——超过损失 30% 一般可认定"过分高于"〔核验条号〕；逾期付款无约定时参照 LPR 并可加计 30%–50%（源自原买卖合同解释规则，现行依据须核验）。
**4. 数据**：LPR（复用 A2）。
**5. 标签**：法律专属；公式 + **裁量区间**——输出必须是三行：约定值、30% 规则调减线、LPR 参照区间，各带依据，绝不出单一"应赔数"。
**6. 优先级：中**。频率高但裁量属性强，产品价值在"区间 + 依据"呈现而非算数，放第二批打磨呈现范式。

### A7 征地补偿计算器

**1. 触发**：土地征收补偿纠纷、行政诉讼。卷宗出现征地公告/补偿安置方案。
**2. schema**：province/county（区片定位，必填，✅）、landArea（面积，亩/㎡ 双单位，必填，✅ 权属证书）、landType（耕地/建设用地/未利用地）、zonePriceOverride（区片综合地价，自动查表可人工覆写）、attachments/greenCrops（附着物与青苗，地方标准表或评估值）。
**3. 法律依据**（须核验）：《土地管理法》（2019 修正）第 48 条：区片综合地价（省级制定，**至少每三年调整或重新公布**）+ 安置补助费 + 地上附着物和青苗补偿（省定）+ 被征地农民社保费用；《土地管理法实施条例》(2021)。
**4. 数据**：各省区片综合地价公告——**县/区片粒度，条目数千级，采集成本全清单最高之一**，且使用频率地域集中。
**5. 标签**：法律专属；需地区参数（重查表）。
**6. 优先级：低**。频率×维护成本比最差；建议按客户所在省按需建表，不做全国铺开。

### A8 继承份额计算器

**1. 触发**：法定继承纠纷、遗产分割。卷宗出现死亡证明 + 亲属关系证明/户籍且无有效遗嘱（或遗嘱仅覆盖部分遗产）。
**2. schema**：estateValue（遗产总值，✅）、isCommunityProperty（是否含夫妻共同财产需先析产，✅ 婚姻状况）、heirs[]（继承人清单：关系、顺位、是否丧失/放弃继承、是否代位/转继承、是否尽主要扶养义务/有扶养能力而不尽义务——后两项为裁量标记）、fetusReserved（胎儿保留份）、estateDebts（遗产债务）。亲属关系图谱可由 LLM 从卷宗构建后**必须人工确认**（错一个节点全错）。
**3. 法律依据**（须核验）：《民法典》第 1122、1127（顺位）、1128（代位）、1130（均等 + 多分少分裁量）、1152（转继承）、1153（先析产）、1155（胎儿保留份）、1159/1161（债务清偿限定继承）条。
**4. 数据依赖**：无。
**5. 标签**：法律专属；纯规则 + 裁量（多分/少分输出为标记与说明，不改份额数）。
**6. 优先级：中**。规则纯、零数据维护，但输入建模（亲属图谱）交互成本高；适合第二批，且是右栏结构化交互（家系树控件）的好素材。

### A9 小额贷款计算器（含 IRR 真实利率还原）

**1. 触发**：金融借款/网贷纠纷；也作 A2 的姊妹件（等额本息分期场景下把"名义月费率"还原为 IRR 年化，判断是否超司法保护上限）。
**2. schema**：principal、installments[]（每期还款额与日期）、nominalRate、feeUpfront（服务费/砍头息）。
**3. 法律依据**（须核验）：IRR 还原本身是金融数学；法律钩子在——金融机构借款利率不适用 4 倍 LPR 上限但过高部分（保险费、服务费等综合成本）可依《最高人民法院关于进一步加强金融审判工作的若干意见》（法发〔2017〕22 号）等调整〔核验：金融借款利率司法保护口径近年变动大〕。
**4. 数据**：无。
**5. 标签**：通用偏法；纯公式。
**6. 优先级：中**。作为 A2 的技术同族顺带实现（同一分期现金流引擎），单独入口价值一般。

### A10 工伤赔偿计算器

**1. 触发**：工伤保险待遇纠纷、劳动争议。卷宗出现工伤认定决定书/劳动能力鉴定结论书即强触发（两份文书几乎提供全部参数）。
**2. schema 草案**

| 字段 | 含义 | 类型 | 必填 | 卷宗可抽取 |
|------|------|------|------|-----------|
| coordinatingRegion | 统筹地区（定社平与省级补助金表） | enum | 必填 | ✅ |
| disabilityGrade | 伤残等级 1–10 / 工亡 | enum | 必填 | ✅ 劳动能力鉴定结论 |
| myWage | 本人工资（受伤前 12 个月平均月缴费工资） | money | 必填 | ✅ 社保缴费记录（**须做 60%–300% 社平封顶托底校验**） |
| isInsured | 单位是否参保（定支付主体拆分） | bool | 必填 | ✅ |
| terminationIntended | 是否解除劳动关系（触发一次性医疗+就业补助金，5–10 级） | bool | 必填 | 人工确认 |
| stopWorkMonths | 停工留薪期 | int | 可选 | ✅ 医嘱 + 地方目录比对 |
| deathCase | 工亡：供养亲属清单（配偶/其他/孤寡孤儿标记） | object[] | 条件必填 | ✅ 亲属关系 |
| accidentYear | 事故年度（定当年全国收入基数） | int | 必填 | ✅ |

**3. 法律依据**（须核验）：《工伤保险条例》（2010 修订）第 33–39 条：一次性伤残补助金月数表（1–10 级：27/25/23/21/18/16/13/11/9/7 个月本人工资）；伤残津贴（1–4 级 90/85/80/75% 本人工资，5–6 级 70/60% 由单位发）；**一次性工亡补助金 = 上年度全国城镇居民人均可支配收入 × 20**（第 39 条）；丧葬补助金 = 6 个月统筹地区上年度职工月平均工资；供养亲属抚恤金（配偶 40%/其他 30%/孤寡孤儿加 10%，合计不超本人工资，人社部令第 18 号）；一次性工伤医疗补助金 + 一次性伤残就业补助金按**省级规定表**（各省月数表差异大）；《社会保险法》第 38–41 条（未参保时用人单位全担）。
**4. 数据**：全国城镇居民人均可支配收入（国家统计局每年 1 月公布，**单一数字、全国统一**，工亡 20 倍基数——维护成本最低的年度数据）；各统筹地区社平工资（年度）；31 省一次性医疗/就业补助金月数表（省级规章，数年一修，静态成本）；各地停工留薪期分类目录（低频）。
**5. 标签**：法律专属；地区参数 + 查表，**几乎无裁量**——确定性在赔偿类里最高。
**6. 优先级：高**。频率高、文书参数齐、结果确定；工亡补助金全国一个数，最小可用版本（工亡 + 1–10 级补助金）数据成本极低。

### A11 五险一金计算器

**1. 触发**：劳动争议中核算未缴/补缴社保损失、入职谈判、法务合规测算。
**2. schema**：city（统筹市，必填）、grossSalary（税前工资/缴费基数，✅）、year（缴费年度）、hukouType（部分城市影响险种）、housingFundRate（公积金比例 5%–12%，✅ 劳动合同/公积金记录）。
**3. 法律依据**（须核验）：《社会保险法》；养老单位 16%/个人 8%（国办发〔2019〕13 号降费率后）；医疗（生育并入）、失业、工伤（行业浮动费率 0.2%–1.9% 基准）各地不同；《住房公积金管理条例》5%–12%。
**4. 数据**：**城市级**基数上下限（社平 60%–300%）与费率表，每年约 7 月调整——**300+ 统筹区，全清单维护成本最高**。
**5. 标签**：通用偏法；重地区参数。
**6. 优先级：低→中**。建议只维护 10–20 个重点城市起步，命中率覆盖多数客户；全国铺开留给"数据维护即场景"的付费/众包机制。

### A12 拆迁安置补偿计算器

**1. 触发**：国有土地上房屋征收补偿纠纷（行政）。卷宗出现征收决定/补偿方案/评估报告。

**3. 法律依据**（须核验）：《国有土地上房屋征收与补偿条例》（国务院令第 590 号）第 17、19、22、23 条；评估依《国有土地上房屋征收评估办法》；集体土地上房屋另循《土地管理法》体系（与 A7 交叉）。
**4. 数据**：地市级搬迁/安置费标准（不定期）、评估依赖外部输入。
**5. 标签**：法律专属；地区参数 + 强裁量/评估依赖——**计算器只能出"框架清单 + 已知项合计"**，不适合出总数。
**6. 优先级：低**。确定性差、数据碎，产品形态更接近"补偿项目核对清单"而非计算器；建议改造为 checklist 型场景卡而非算数卡。

### A13 物业费计算器

**1. 触发**：物业服务合同纠纷（追缴物业费）。批量小额案件（法务/物业公司客户）有批处理价值。
**2. schema**：area（建筑面积，✅ 合同/产证）、unitPrice（合同单价，✅）、period（欠费期间，✅ 催缴记录）、lateFeeClause（违约金条款，✅）、reductionClaims（服务瑕疵抗辩标记，裁量）。
**3. 法律依据**（须核验）：《民法典》第 944 条；《最高人民法院关于审理物业服务纠纷案件的司法解释》（法释〔2020〕改正版）；部分地区政府指导价目录。

## 三、B 区详述 · 诉讼程序（4 项）

### B1 诉讼费计算器

**1. 触发**：任何案件立案前（诉讼费预估进《起诉状》配套材料）、结案时（费用负担核算）。**应做成所有诉讼场景的默认挂件**而非独立入口——起诉状生成场景（S 系）产出时自动附带。
**2. schema**：caseType（财产/离婚/人格权/其他非财产/知产/劳动/行政，必填，✅ 案由映射）、claimAmount（诉讼标的额，必填，✅ 诉讼请求）、procedure（普通/简易/小额/二审/再审，必填）、feeEvents[]（申请费事件：执行/保全/支付令/公示催告，可选）、settlement（调解/撤诉减半标记，可选）。
**3. 法律依据**（须核验）：**《诉讼费用交纳办法》（国务院令第 481 号，2007-04-01 施行）**第 13 条财产案件分段累进：≤1 万元 50 元；1–10 万 2.5%；10–20 万 2%；20–50 万 1.5%；50–100 万 1%；100–200 万 0.9%；200–500 万 0.8%；500–1000 万 0.7%；1000–2000 万 0.6%；2000 万以上 0.5%（**各段为"超过部分"累加**）。离婚 50–300 元 + 财产超 20 万部分 0.5%；人格权 100–500 元 + 超 5 万至 10 万 1% + 超 10 万 0.5%；其他非财产 50–100 元；知产无争议金额 500–1000 元；劳动争议 10 元；行政（商标/专利/海事 100 元，其他 50 元）。第 14 条申请费：执行（≤1 万 50 元；1–50 万 1.5%；50–500 万 1%；500–1000 万 0.5%；>1000 万 0.1%）；保全（≤1000 元或不涉财产 30 元；1000 元–10 万部分 1%；超 10 万部分 0.5%，**上限 5000 元**）；督促程序按财产案件受理费的 1/3；公示催告 100 元。简易程序/小额减半收取，调解结案或撤诉减半（第 15、16 条）。
**4. 数据**：**费率表 2007 年施行至今未修改——纯静态表，一次建成零维护**〔核验：个别地方性减免政策〕。
**5. 标签**：法律专属；纯查表 + 分段公式。
**6. 优先级：高（第一批首选）**。零数据维护、百分百确定、每案必用、实现半天——投入产出比全清单第一。

### B2 刑期计算器

**1. 触发**：刑事辩护场景（Stage 2 刑事线）：量刑预测辅助、数罪并罚核算、羁押折抵、减刑假释资格日推算。竞品高亮位，民刑兼顾的差异化入口（docs/48 注记）。
**2. schema**：sentences[]（各罪刑种 + 刑期）、detentionStart/JudgmentDate（先行羁押起止，✅ 拘留/逮捕证 + 判决日）、probation（缓刑标记）、combinePrinciple（并罚场景）、commutations[]（历次减刑裁定，✅）。
**3. 法律依据**（须核验）：《刑法》第 69–71 条数罪并罚（限制加重：总和刑以下、最高刑以上；有期总和不满 35 年顶格 20 年、35 年以上顶格 25 年；管制顶格 3 年、拘役顶格 1 年）；第 41、44、47 条刑期折抵（羁押 1 日折有期/拘役 1 日、折管制 2 日）；第 73 条缓刑考验期（拘役：原判以上 1 年以下不少于 2 个月；有期：原判以上 5 年以下不少于 1 年）；第 78–86 条减刑假释（假释：有期 1/2、无期 13 年）及《最高人民法院关于办理减刑、假释案件具体应用法律的规定》（法释〔2016〕23 号及后续补充）减刑幅度与间隔。
**4. 数据**：无（法条级，修法才变）。
**5. 标签**：法律专属；纯规则公式（组合复杂但无地区参数、无裁量——量刑本身不算，只算刑期算术）。
**6. 优先级：中（当前）/ 高（Stage 2 刑事线开启时）**。零维护成本，可提前实现压库存。

### B3 立案审理期限计算器

**1. 触发**：每个在办案件的日程底座——与 docs/48"诉讼日程与期限族"直接合流：立案日一录入，审限到期日自动进案件日历（高可靠等级、零 LLM 通道）。
**2. schema**：caseTrack（民/刑/行）、procedure（普通/简易/小额/特别/二审判决/二审裁定）、filingDate（立案日，✅ 受理通知书）、excludedPeriods[]（不计入审限事件：公告、鉴定、管辖权异议、中止等，✅ 各程序性文书，**每项须人确认**）、extensions[]（延长审批记录）。
**3. 法律依据**（须核验）：《民事诉讼法》：一审普通 6 个月（可延 6 个月 + 报上级批）、简易 3 个月（可延 1 个月）、小额 2 个月、二审判决 3 个月/裁定 30 日、特别程序 30 日〔核验 2023 修正条号〕；《刑事诉讼法》第 208 条（一审 2 个月至迟 3 个月 + 各类延长）、二审 2 个月；《行政诉讼法》第 81、88 条（一审 6 个月、二审 3 个月）；**《最高人民法院关于严格执行案件审理期限制度的若干规定》（法释〔2000〕29 号）第 9 条不计入审限项**〔核验后续修订〕。
**4. 数据**：法条级 + 法定节假日表（期间顺延用，国办年度通知）。
**5. 标签**：法律专属；查表 + 日期公式。
**6. 优先级：高**。与时效族共用期间引擎，是日程场景的确定性内核，不是孤立计算器。

### B4 公告/开庭日期推算器

**1. 触发**：被告下落不明需公告送达的案件；律师排庭期预估。
**2. schema**：serviceType（国内公告/涉外公告/直接送达）、announcementDate（公告发出日，✅ 公告回执/报纸）、isForeign（涉外，✅ 当事人住所）、defensePeriod（答辩期 15 日/涉外 30 日）、evidencePeriod（举证期限，法院指定值，✅ 举证通知书）。
**3. 法律依据**（须核验）：《民事诉讼法》（2023 修正，2024-01-01 施行）：国内公告送达自发出之日起经过 **30 日**视为送达；**涉外公告送达 2023 修正后缩短为 60 日**（原 3 个月）——已经外部信源核实（最高法国际商事法庭对修正条款的解读及 2023 修正决定文本）；答辩期 15 日/涉外 30 日；举证期限普通程序不少于 15 日（《民事诉讼证据规定》法释〔2019〕19 号）；期间计算依《民诉法》第 85 条 +《民法典》第 200–204 条（起算日不计入、末日节假日顺延）。
**4. 数据**：法定节假日表（年度）。
**5. 标签**：法律专属；纯日期公式。
**6. 优先级：高**。与 B3 共用同一期间引擎，边际成本近零，一起上。

## 四、C 区详述 · 通用工具（16 项）

C 区在竞品处是流量件（chatbot 需要工具箱撑门面）；在 Courtwork 的入口消失范式下，**没有法律钩子的通用件不值得独立场景位**，只保留三类：(a) 有法律语义的升级件；(b) 法律场景的内嵌子件；(c) 明确不做。

### C5 日期计算器 → 升级为「法律期间计算器」（有法律语义，高优先）
通用日期加减对律师无差异价值；升级点：**按《民法典》第 200–204 条与《民诉法》第 85 条计算法律期间**（起算日不计入、按月按年对应日、末日为法定节假日顺延至次日、期间届满时刻），覆盖上诉期（判决 15 日/裁定 10 日）、举证期、申请执行时效（2 年）、诉讼时效（3 年 + 中断中止标记）。数据：国办法定节假日年度通知（**每年 11–12 月发布次年安排，含调休——调休表是顺延判断的必需数据**）。确定性：纯公式 + 节假日表。它是 B3/B4 与时效族共用的**期间引擎本体**，优先级高。

### C9 人民币大写转换（有法律语义，零成本高优先）
法律文书（起诉状、合同、收据）刚需；纯规则（会计写法规范：零的进位处理、"整"字、"元角分"）；无数据依赖。不设独立入口，做成 output 管线的文书内联函数 + 场景内小工具。优先级高（实现成本约一小时）。

### C1 个税计算器（通用偏法，中优先）
法律钩子：劳动争议中经济补偿金税后净额（财税〔2018〕164 号：3 倍社平以内免征，超出部分单独适用综合所得年度税率表〔核验政策延续〕）、劳务报酬/稿酬预扣、年终奖单独计税（延续至 2027 年底，财政部税务总局公告 2023 年第 30 号〔核验〕）。依据：《个人所得税法》（2018 修正）七级超额累进表 + 专项附加扣除（国发〔2023〕13 号提标：子女教育/婴幼儿照护各 2000 元/月、赡养老人 3000 元/月〔核验〕）。数据：税率表稳定、扣除政策年度检查。建议作 A4 的税后视图子件而非独立入口。

### C2/C3/C4 贷款/房贷/存款利息（通用工具，低优先）
等额本息/等额本金/单利均为纯公式，与 A2/A9 共用分期现金流引擎。房贷需 LPR（复用 A2 表）+ 公积金利率现值（2025-05-08 起首套 5 年以上 2.6%、5 年内 2.1%；二套 3.075%/2.525%；央行公告不定期〔已网搜核实〕）。法律钩子弱（离婚析产时算剩余本息有用），不设独立入口，作离婚/执行场景子件。

### C10 单位转换（低优先，子件）
法律有用的只有亩↔平方米（1 亩 = 10000/15 ≈ 666.67 ㎡）与公顷，作 A7/A12 内嵌换算，不独立。

### C6/C7/C8/C11/C16 阴阳历/邮编/区号/数学计算器/世界时间（不做/最低）
纯 chatbot 流量件，与案件文件夹范式无交集；LLM 对话本身可答（或系统计算器可代），铺入口是负资产。裁决：**不做**。若坚持要（送达地址核验用邮编），用静态开源库随手挂，不占场景位。

### C12*/C13*/C14*/C15* 汇率/契税房产税费/购置税/养老金（推测项）
- 汇率：涉外案件标的折算有微弱钩子（判决日中间价，外管局/央行日更）——按需内嵌，不独立。
- 契税/二手房税费：房产纠纷与析产场景有钩子（契税 1%–3% + 地方优惠、增值税 5% 满二免、个税 1%/20% 差额〔核验，政策变动频繁〕）——低优先，政策波动大维护烦。
- 车辆购置税（10%，新能源减免政策期〔核验〕）：不做。
- 养老金测算：城市级计发基数年度更新，维护成本类 A11 而使用频率更低：不做（留给 A11 扩展）。

## 六、对 Courtwork 的上架批次建议

### 第一批（MVP 后首个场景包，建议 7 项）

| 项 | 入选理由 |
|----|----------|
| B1 诉讼费 | 零维护 + 每案必用 + 实现最简，作为 S10 场景族的握手样板（schema 握手协议 docs/53 的最小实证） |
| A5 迟延履行利息 | 零维护、执行案件量大、公式全确定 |
| A2 民间借贷利息 | 最高频案由之一；LPR 表建成即长期资产；分段流水引擎是 A6/A9/C2 的共用内核 |
| C5 法律期间引擎（含 B3 审限 + B4 公告推算） | 与时效族（docs/48）共用零 LLM 期间内核，一次实现三处受益 |
| A4 劳动工资三件套 | 参数全可卷宗抽取，"LLM 填表员"范式演示性最佳 |
| A10 工伤赔偿（最小版：工亡 + 一次性伤残补助金） | 工亡基数全国单值、月数表静态，数据成本极低而结果冲击力强 |
| C9 人民币大写 | 约一小时成本，文书管线内联即赚 |

第一批共同特征：**数据依赖 ≤ 3 张表（LPR、节假日、全国收入单值），全部可在一周内建成**；地区参数重的（A1 人损）刻意推迟到省级三指标表建完再上。

### 第二批（省级参数表建成后）
A1 人身损害（旗舰项，值得配右栏分项明细控件精做）、A3 非因工死亡（数据复用 A1）、A6 违约金（裁量区间呈现范式）、A8 继承份额（家系树控件）、A10 工伤全量（省级补助金表）、C1 个税（作 A4 税后视图）、B2 刑期（压库存，随 Stage 2 刑事线转高亮）。

### 缓建/不做
A7 征地、A12 拆迁（数据成本/频率比最差，按客户省按需）；A11 五险一金（先 10–20 城）；A13 物业（改 checklist + 批量模式再评估）；C6/C7/C8/C11/C14/C15/C16 纯流量件**不做**——竞品铺 33 个入口是 chatbot 范式的门面需求，我们按案由自动铺表，空入口是负资产。

### 三条实施纪律（对齐总纲）
1. **schema 先行**：每个计算器的输入/输出 schema 落 `packages/schemas`（输出统一为 `CalcArtifact`：分项明细行[金额+依据条款+参数溯源 SourceAnchor]+适用标准版本号+裁量区间标记），脚本落独立纯函数包，golden files 用真实费率表算例快照。
2. **法条引用须过核验门**：本文全部条款号以调研者知识写成，实施每一项前逐条对国家法律法规数据库核验现行文本与条号（民诉法 2021/2023 两次修正致条号漂移，尤其注意）。
3. **裁量项永不出单值**：A1 精神抚慰金、A6 调减、A8 多分少分、A12 停产停业损失一律输出"区间 + 依据 + 待律师裁断"标记——留人确认是产品纪律，不因是计算器例外。
