## 批次五开册（2026-07-11，AUDIT-1 穷举式 UI 自查）

状态：**待架构裁决；本批仅登记差异，未改产品代码。** 普查以 `docs/46` 已登记控件和 `docs/57` 的 19 个 schema 原语为行，以 default / hover / active / disabled / empty / loading / error 为状态列；对明确不做、N/A、待规格且尚未上屏的项目不伪造状态。独立端口真机逐格留存 45 张截图，覆盖欢迎与授权、五工作面、命令面板、设置六组、新建/归档、专注/对照/双栏折叠、composer 溢出、整理计划与报告、空案件等路径。

机器底册：既有 `lint:motion`、`lint:signature`、`lint:icons`、`lint:elevation` 全绿。DSGN-1 站点版 deslop 规则首跑的原始命中为 raw hex 105、shadow 3、gradient 7、purple 0、sub-12 11、large-radius 0、layout-animation 1；复核表明 raw hex 主要来自 token 声明与 icon audit fixture，gradient 来自滚动遮罩、品牌 hover 折射及用量环，layout-animation 是图谱配置 `animation: false` 的字符串误报。产品化扫描器需先加入 token/fixture/已批准角色白名单；本轮可行动的机器命中是 sub-12。修订风险选中态的首帧疑点经 250ms 复核为 `rgb(233, 238, 244)`，符合 `bg.selected`，不登记为差异。
