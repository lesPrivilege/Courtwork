# 情况说明

出具方：起云智能装备（虚构）有限公司

关于合同编号LJJZ-CY-2024-0817项下设备验收及付款事宜，现说明如下：

一、2024年12月10日设备联调验收当日，我司现场技术人员即发现熔炼炉温控系统存在异常报警情况，经与临江精铸现场代表梅昭阳口头沟通后，对方称属调试阶段常见现象，我司在未充分核实的情况下签署了验收单。

二、验收后我司生产部门持续反馈该设备温控运行不稳定（2024年12月12日、12月15日），我司于2024年12月20日与临江精铸召开会议就此进行协商（详见会议纪要一），会议中我司已明确提出验收当日设备异常、尚未正式验收的立场。

三、我司于2025年2月5日提出委托第三方机构对该设备进行检测，以核实设备是否符合合同约定的技术参数要求，但双方未能就检测费用分摊达成一致，检测工作暂未实际进行。

四、基于上述情况，我司认为在设备质量问题未澄清前，全额支付验收款存在合理异议，现将有关情况说明如上，供各方及后续处理参考。
