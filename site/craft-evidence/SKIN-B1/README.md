## 客观差异（逐帧像素采样，非主观描述）

主色占比在两版之间**完全一致**，只有色值改变——这是「只换色不动版」的机器可核证据：

| 帧 | before | after | 占比 |
|---|---|---|---|
| 01 | `#FFFFFF` / `#F6F9FC` / `#EAEFF4` / `#DDE7F2` | `#FFFFFF` / `#F7F8FA` / `#F2F4F7` / `#D9E3F6` | 37% / 28% / 15% / 2%（两版相同） |
| 03 | `#F6F9FC` / `#FFFFFF` / `#EAEFF4` / `#DDE7F2` | `#F7F8FA` / `#FFFFFF` / `#F2F4F7` / `#D9E3F6` | 63% / 14% / 13% / 1%（两版相同） |

占比不变 = 版面分区面积不变；色值成对位移 = 只发生 token 层置换。若版式被动过，占比必然漂移。
