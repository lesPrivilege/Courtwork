## 回报整合区（2026-07-09 已整合）

**采纳**：Stripe 藏青黑 `#0A2540` 正文色（拒纯黑与暖灰）；tabular-nums 等宽数字（法条编号/金额/日期对齐，硬性规范）；Granola 式 1px 全站网格线（条理感外化）；Linear 式 KBD 快捷键提示常驻；Stripe 状态徽章工艺；中西文混排字距与 justify 排版参照（Gemini 报告附完整 CSS 规格卡，作为 tokens 初稿基调）。

**规避（采信 Gemini 反方理由）**：Vercel 无边框（法条雪崩，律师需信源容器边界）；Raycast 12px 极限字阶（目标用户 35–55 岁，正文锁 14–15px / 行高 1.55–1.6）；全站 0 动效（2–5 秒 AI 运算期会被读成死机——**长任务微反馈动效为必做组件**：边框微光流转/骨架呼吸）。

**否决**：Grok 的"专业暗黑"基调——浅色优先是既有拍板（律所办公环境），暗色不进 MVP。Grok 其余建议（Tauri polish、全状态覆盖、tokens 治理）与既有决定重合，视为背书。

**签名动作拍板：「法理之线」**（Gemini 候选 1）——内容卡片左侧常驻 2px 冷灰竖线，状态变更硬切语义色（高危红 `#DC2626` / 修订蓝 `#2563EB` / 权威绿 `#16A34A`）+ 顶部 12px 等宽字母徽章。理由：语义色预算的物理载体 + 纸质卷宗侧边朱批的本能隐喻。候选 3（全局刻度尺 minimap）降为长文档组件进 backlog（S3/S5 长合同审阅）；候选 2（抽屉）吸收为右栏面板交互模式参考。信源 A/B/C 角标、风险高中低、修订增/删的完整语义色映射由 Design 第一步定稿。
