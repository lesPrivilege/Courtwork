```text
TypeError: Cannot read properties of undefined (reading 'getData')
at _Minimap.getElements (.../chunk-TFT3KDDL.js:88790:30)
at _Minimap.renderMinimap (.../chunk-TFT3KDDL.js:88783:24)
```

URL 为 `http://127.0.0.1:1421/node_modules/.vite/deps/chunk-TFT3KDDL.js`，记录时间 `2026-07-13T03:04:53.748Z`。fresh tab 的单一路径复核未每次复现，但完整路径中已独立捕获未处理 TypeError；视觉上未出现白屏，仍按运行时阻断记账，不自行降级。

#### `LUNA-UI-002 ❓`：无 key 发送的引导层归属

Chat 中 Send 被拦截且草稿保留，功能语义符合“未连接不得发”；实际视觉落点是全局 Settings/Manage credentials，而不是独立的 chat provider guide。需架构确认两者是否同一既裁语义，验收不代判。

#### `LUNA-UI-003 ❓`：环境网络遥测错误

点击工作操作菜单时工具侧捕获 Statsig POST timeout（`https://ab.chatgpt.com/v1/rgstr`）。该错误来自浏览器环境外部网络，不等同于 Courtwork 业务异常；仍保留编号，待架构决定是否从本地验收噪声中排除。

#### `LUNA-UI-004 ❓`：Paste text 无输入时无可见落点

`Add → Paste text` 菜单项可点，点击后菜单关闭；本轮没有注入剪贴板文本，未观察到新附件/消息。无法区分“空剪贴板的正确无事发生”与“缺少反馈”，待真机带剪贴板复核。
