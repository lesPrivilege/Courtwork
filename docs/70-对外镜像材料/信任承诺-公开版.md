# 数据与信任承诺（公开版）

1. **案件内容永不用于训练。** 这不是政策选择，是法律工作的必然——并且写进架构：不存在把案件内容送入训练管道的代码路径。
2. **本地优先。** 数据存在你的电脑上，按案件分目录物理隔离；跨案件读取在数据访问层结构性不存在（利冲合规的存储学表达）。API 密钥只存系统钥匙串，界面层无明文读取通道。
3. **原件只读。** 上传的卷宗原件永远不可被编辑——证据不可篡改是执业纪律，也是我们的代码纪律。
4. **诚实状态。** 连接状态只以真实请求成功为准；材料不足的字段标"待补"并说明缺什么，模型不编造；本地存储不等于加密，我们不把"本地"说成"绝对安全"。
5. **留人确认。** 写文件、对外提交、改变权威记录——一切不可逆动作停在确认门禁，由你决定。
