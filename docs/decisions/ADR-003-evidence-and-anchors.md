# ADR-003：信源、引用与锚点

- 状态：Accepted
- 日期：2026-07-13
- 来源：`99b85b3`、`161804a`、`aa914da`、`7f98c30`

## 决定

信源分为：

- A：权威确定性接口，可进入正式事实通道；
- B：自建或演示库，必须携覆盖边界；
- C：开放网络，只能作为 `verified:false` 的参考线索。

web search 是普通工具，不是 core 特权。等级传播和门禁由系统执行，场景只声明哪些等级可进入哪些槽位。

模型只生成 quote；系统在版本化阅读视图中寻找唯一匹配并铸造 SourceAnchor。无法唯一定位时允许一次受限修复；仍失败则剪枝为 coverage 缺口。正式 docx 批注不得使用未经确认的 C 级事实。

## 后果

- citation 展示文本不是外键；门禁只认 evidence key 与锚点；
- `quote === source slice` 必须可机器验证；
- 锚点属于材料版本，不随 turn 生命周期变化。
