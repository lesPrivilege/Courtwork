# 品牌图标

权威源稿为 `icon-dark.svg` 与 `icon-light.svg`，几何完全一致，只交换背景与前景关系。

母题是“线 + 文”：一条竖线立在三行抽象文书旁，表达有边、有据、有人把关。禁止改成天平、法槌、盾牌、立柱或书卷等法律陈词滥调。

## 规则

- 品牌图标固定使用藏青与中性色，不随业务状态换色。
- 16px 必须仍读作“线 + 文”，32px 可辨三行结构。
- UI 内品牌 mark、思考反馈与应用图标可以共享几何母题，但不得借用法理之线的红/琥珀/蓝/绿处置语义。
- 新平台尺寸从两份源稿派生，不创建第三套配色或新母题。

## 变体谱系（记谱，2026-07-19 立）

母题的**唯一几何真源**是 512 源稿（`icon-dark.svg` / `icon-light.svg`）的 5/6-rect 体系：
满幅底盘（仅应用图标保留）+ 竖线 + 三行文书（长/长/短）。其余尺寸一律是该体系的**登记变体**，
从源稿重推导，不得另起一套几何，也不得把旧线稿重上色充数——**「一致性」的口径是几何 + 单源，
不是仅颜色**。变体入本表登记后方可使用。

| 变体 | 用处 | 相对源稿的简化 | 单源核验 |
|---|---|---|---|
| `site/assets/icon.svg`（24 网格，4 rect） | Pages 站 wordmark 标记（18px 显示）与 OG 卡记号 | ①去底盘（wordmark 规则既有）；②单色——源稿竖线与文书行分两级灰，18px 下两级会糊成一团，故取单色；③圆角按比例收并对齐 24 网格；④文书行三宽度取整（.686→.667） | deslop `brand-lineage` 门：从源稿**现算**四枚 rect 的归一化比例再核变体，容差 .03。源稿改了变体不跟即红 |

**小尺寸简化是变体的义务，不是偷工**：直接把 512 源稿缩到 24 会让圆角与两级灰在小尺寸糊成
一团。简化项须逐条记入上表，未登记的简化等于私自分叉。

同一枚变体在同一产物内**只能有一份**：OG 卡的大记号与 wordmark 小记号共用
`site/assets/icon.svg`，不得用 CSS 方块手工复刻母题（复刻＝第二份几何真源，源稿一改即悄悄脱钩）。

## 图标与文字锁定标记

- 应用图标和系统发布资产可以按平台规范保留方形底盘；这是安装物图标，不是导航 wordmark。
- 导航、CaseRail 与页面页眉中的文字锁定标记只取“竖线 + 三行文书”的核心几何，透明底、无外框、无圆角底座、无阴影。
- 核心标记固定放在 `Courtwork` 文字左侧，视觉间距 7px 左右并垂直居中；不得放到文字右侧，也不得用悬浮或磁吸动画强调。
