# 债权转让通知书

致：起云智能装备（虚构）有限公司

我司（临江精铸科技有限公司）与华瑞商业保理（云章）有限公司于2025年3月20日签订《应收账款转让协议》，将我司对贵司享有的合同编号LJJZ-CY-2024-0817项下应收账款（验收款2,280,000元及质保金380,000元，合计2,660,000元）转让予华瑞商业保理（云章）有限公司。

根据《中华人民共和国民法典》第五百四十六条规定，债权人转让债权，应当通知债务人，该转让自通知到达债务人时对债务人发生效力。现特此通知，自本通知送达之日起，贵司应向华瑞商业保理（云章）有限公司履行上述款项的支付义务。

特此通知。

临江精铸科技有限公司（盖章）
日期：2025年3月20日

（附）起云智能装备（虚构）有限公司回函确认收悉，日期：2025年3月25日
