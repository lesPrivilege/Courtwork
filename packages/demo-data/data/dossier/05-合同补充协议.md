# 《精密铸造生产线设备采购合同》补充协议

原合同编号：LJJZ-CY-2024-0817
签订日期：2024年9月2日

甲方（卖方）：临江精铸科技有限公司
乙方（买方）：起云智能装备（虚构）有限公司

鉴于乙方厂房改造工程延期，双方经协商一致，就原合同交付期限条款补充约定如下：

**一、** 原合同约定设备应于2024年11月15日前分批交付完毕，现双方一致同意，第三批设备（自动化上下料系统）交付期限顺延至**2024年11月30日**。第一批（中频熔炼炉）、第二批（精密铸造模具及配套系统）交付期限不变。

**二、** 因本次顺延产生的仓储、保管费用由乙方承担，甲方不因此顺延承担逾期交付违约责任。

**三、** 原合同其余条款不变，本补充协议与原合同具有同等法律效力，作为原合同不可分割的组成部分。

**四、** 本补充协议一式肆份，双方各执贰份，自双方签字盖章之日起生效。
